RP

public struct RP : CustomStringConvertible, Equatable

RP instances contain reduction polynomials like x^m + x^k3 + x^k2 + x^k1 + 1 where m > k3 > k2 > k1 > 0 or x^m + x^k1 + 1 where m > k1 > 0
